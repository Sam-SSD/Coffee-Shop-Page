## Estilos y Diseño

### Sistema de Colores
- Primario: #8B4513 (Marrón café)
- Secundario: #D2B48C (Beige)
- Acento: #FFD700 (Dorado)
- Texto: #333333 (Gris oscuro)
- Fondo: #FFFFFF (Blanco)

### Tipografía
- Títulos: 'Playfair Display'
- Cuerpo: 'Roboto'
- Iconos: Font Awesome

### Diseño Responsive
- Breakpoints:
  - Móvil: < 768px
  - Tablet: 768px - 1024px
  - Desktop: > 1024px
